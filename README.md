## :bookmark: Sobre

O <strong>Happy</strong> é uma aplicação Web e Mobile com o objetivo de conectar as pessoas aos orfanatos, de modo que elas encontrem orfanatos na própria cidade ou região e obtenham novas experiências ao entrar em contato com essas crianças levando mais felicidade a elas.

## :rocket: Tecnologias
- [TypeScript](https://www.typescriptlang.org/)
- [Node.js](https://nodejs.org/en/)
- [ReactJS](https://reactjs.org/)
- [React Native](https://reactnative.dev/)

## :fire: Pré-requisitos
- [Node.js v12.18.3](https://nodejs.org/en/)
- [NPM v6.14.6](https://www.npmjs.com/) ou [Yarn v1.22.4](https://yarnpkg.com/)
- [Expo](https://expo.io/)
- [Visual Studio Code](https://code.visualstudio.com/)

## :zap: Como usar
- Faça um clone desse repositório
- Instale as dependências usando: `yarn install` ou `npm install`
- Inicie a aplicação:

  - Servidor
    ```sh
    $ cd backend
    ```
    - Rode o comando de inicialização: 
    `yarn start` ou `npm start`

  - Web
    ```sh
    $ cd web
    ```
    - Rode o comando de inicialização: 
    `yarn start` ou `npm start`
